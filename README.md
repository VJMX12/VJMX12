#### 🔧 What I'm working on

Combines a sticky HMM for regime detection, Bayesian online changepoint detection, a calibrated online learner, and kNN retrieval over past trades to size and gate every decision.
- **[VJAnalysisBot](https://github.com/VJMX12/VJAnalysisBot)** — A crypto analysis toolkit that explains every signal step-by-step (chain-of-thought, not a black box), with backtesting across multiple periods and a Telegram bot front-end.
